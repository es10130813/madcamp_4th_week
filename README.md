## c. 게임 소개
Slime Slayer는 isometric 뷰를 사용한 2D 플랫포머 게임입니다. 총 세 개의 스테이지로 이루어져 있으며, 스테이지를 완료하여 더 강한 무기를 획득할 수 있습니다.
